一、简介
CeleDial 是 Windows Mobile Pocket PC Phone 上必备的智能拨号助手，它可以帮助您简单、方便、快速地定位联系人，让您随心所欲，想拨就拨。

[目录]

二、特点
  1. 迅捷易用：无缝集成到拨号界面，使用电话程序中的 T9 屏幕键盘，简单易用；内存占用极少，运行飞快。
    　
  2. 智能匹配：支持通过姓名、拼音、简拼和电话号码智能查找联系人；支持所有 Unicode 中文汉字拼音。
    　
  3. 码表定制：内置支持简体拼音、通用拼音、注音符号方式搜索联系人，可以通过外置码表支持其它查询方式。
    　
  4. 定制命令：用户可定制的快捷菜单，支持自定义命令扩展，可应用于ＩＰ电话、视频拨号等其它扩展应用。
    　
  5. 广泛支持：支持读取内置联系人、SIM 卡联系人和通话记录电话号码作为 CeleDial 联系人条目。
    　
  6. 完美兼容：支持 Windows Mobile 5.0/6.0/2003 Pocket PC Phone，支持 QVGA、VGA 和正方形屏幕 。

三、安装
  1. 安装：把 CeleDial.cab 复制到设备中，运行即可安装。安装过程中将会强行中止电话程序，安装完成后启动电话程序即可生效。最好安装在设备里，否则启动时无法自动加载。如果想使用其它面板软件，或者 CeleDial.CAB 无法使用，可使用 EXE 版本 CeleDial，完全没有任何兼容性问题。
    　
  2. 卸载：执行“程序→设置→系统→添加删除程序”，在对话框中选中 CeleDial，然后选择 卸载 即可。Pocket PC 2003 中如果不能卸载，请先执行“CeleDial 设置...→菜单→退出”命令，然后再卸载；或者把安装文件夹下的 CeleDial.dll 改名，卸载后，手动删除 CeleDial 安装文件夹。

四、使用
  1. 进入电话程序，用电话程序中的 T9 屏幕键盘输入你要查找的联系人，CeleDial 会按您的输入意图列出匹配的联系人。 您可以用下面几种方式查找联系人：
    　
    • 电话号码：如输入 13957，将会匹配 13957162305 条目；如果在选项中启用“任意号码”，则输入 6230 也可匹配。
      　
    • 姓名拼音：如输入 qiuchuji（对应 T9 键盘输入 74824854），将会匹配 丘处机 条目；如果在选项中启用“任意拼音”，则输入 chuj 也可匹配。
      　
    • 姓名简拼：如输入 sbe（对应 T9 键盘输入 723），将会匹配 孙不二 条目。最后一个汉字也可以输入全拼，如 sber，以提高精确度。
      　
  2. 选中匹配条目后，可执行以下操作：
    　
    • 按 回车 键拨打该条目显示的电话号码。如果在选项中启用“使用拨号按键”，则使用拨号键也可以执行此动作；否则，拨号键用来拨打你输入的号码，而不是 CeleDial 的条目。
      　
    • 如果有多个电话，可用向左或向右键切换显示号码。
      　
    • 用触笔长按条目，弹出菜单，然后从菜单中选择其它号码，或执行文字短信、ＩＰ拨号等其它操作。请参看“命令列表”。

五、菜单
  1. 在电话程序中，执行“菜单→CeleDial 设置...”命令，可进入 CeleDial 设置属性页，此时屏幕下部就是 CeleDial 菜单。
    　
  2. 在 CeleDial 窗口紧接着的下面位置（即输入显示号码的地方），长按触笔，亦可弹出 CeleDial 菜单。
    　
    • 刷新：刷新联系人列表。
      　
    • 选项：显示“选项”属性页。
      　
    • 命令：显示“命令”属性页。
      　
    • 致谢：显示“致谢”属性页。
      　
    • 帮助：显示“帮助”属性页。
      　
    • 关于：显示“关于”属性页。
      　
    • 退出：终止电话程序进程（EXE 版本仅退出 CeleDial 程序）。

六、设置
  1. 选项：指定程序运行中的可选设置。
    　
    • 启用智能拨号：指定是否启用 CeleDial。可在此暂时关闭智能拨号功能。
      　
    • 拦截拨号按键：使用拨号键拨打 CeleDial 条目，否则拨号键用来拨打输入的号码（即屏幕显示的号码）。如果启用，则需要使用屏幕上的“通话”按钮来拨打输入号码。
      　
    • 显示条目序号：在每个联系人条目前显示序号，此选项可统计匹配条目的个数。
      　
    • 匹配姓名拼音：使用姓名的拼音查找联系人。由于英文联系人的拼音就是英文本身，所以此法亦可用于匹配英文联系人。
      　
    • 匹配姓名简拼：使用姓名的简拼查找联系人。最后一个汉字也可以输入全拼（混拼），以提高精确度。
      　
    • 匹配电话号码：使用电话号码查找联系人。CeleDial 支持联系人中的 8 种电话号码。
      　
    • 任意拼音：任意位置的姓名拼音匹配。若不选择，仅从第一个字符开始匹配。
      　
    • 任意简拼：任意位置的姓名简拼匹配。
      　
    • 任意号码：任意位置的电话号码匹配。
      　
    • 读取联系人：读取内置联系人列表。
      　
    • 读取 SIM 卡：读取 SIM 卡上的联系人。按序号读取，若该序号空闲，则停止读取，所以请保持联系人 SIM 卡联系人连续储存，中间位置不要空闲。
      　
    • 读取通话记录：读取通话记录作为联系人。CeleDial 会自动过滤重复的通话记录，且通话记录数目有变化时，会自动刷新通话记录。
      　
  2. 命令：查看或编辑 CeleDial 联系人条目弹出菜单中的命令列表。
    　
    • 添加：添加一个命令条目。
      　
    • 删除：删除一个命令条目。
      　
    • 上移：把命令条目向上移动一行。
      　
    • 下移：把命令条目向下移动一行。
      　
    • 命令列表：列出了选定联系人条目弹出菜单的命令。可以添加或删除命令条目，也可以双击编辑条目。
      　
      • 出现：命令在弹出菜单中的最大出现次数（针对一个号码出现一次），0 则不显示此命令。
        　
      • 名称：命令名称。名称和命令均支持宏扩展：

        %n 姓名
        %y 拼音
        %j 简拼
        %p 电话
        %t 类型
        %a 所有电话
        　
      • 命令：要执行的命令。支持三类命令：

        1). 如果有“.exe”，则执行 EXE 程序（请参照短信条目）。
        2). 如果有冒号“:”，则调用外壳执行它，比如“mailto:%p?body=%y”，将会调出电子邮件程序。
        3). 如果前两项都不是，则直接拨打命令内容，比如“17951%p”。
        　
  3. 致谢：显示捐赠者名单和其它致谢信息。
    　
  4. 帮助：显示此帮助信息。如果帮助文件丢失，可连接到 Internet 上查看帮助。
    　
  5. 关于：显示程序信息、版本号和版权信息。

七、技巧
  1. 如何更改 IP 拨号的前缀？

    在电话程序中执行“菜单→CeleDial 设置...→命令”，在命令列表中双击“ＩＰ拨号”的条目，然后修改拨号格式即可。
    　
  2. 我的设备安装 CeleDial.cab 后没有任何效果，怎么办？

    部分设备（如惠普的某些 Pocket PC Phone）安装 CeleDial.cab 可能没有效果，因为它的电话程序没有载入 CeleDial.dll。但你可以安装 EXE 版本 CeleDial（把 CeleDial.exe 复制到 \Windows\Startup 目录中），功能上和 DLL 版本完全一致。EXE 版本不能与 DLL 版本同时安装。
    　
  3. 我的设备是正方形屏幕的，只能显示一条 CeleDial 联系人；或者我是普通屏幕的设备，但我希望在一屏上显示更多的联系人，怎么做？

    在 CeleDial 窗口紧接着的下面位置，即输入显示号码的地方，在此位置单击，可扩展 CeleDial 窗口的大小，以便显示更多的联系人；再次点击该处可收缩回原位置。另外，在此处长按触笔，可弹出 CeleDial 菜单。
    　
  4. 怎样显示所有联系人？

    CeleDial 支持星号 * 通配符，一个 * 代表一个电话数字（通配符只能匹配电话号码，不匹配姓名拼音或简拼）。输入星号 *，可匹配所有联系人，但最多只能显示 999 条。
    　
  5. 有办法快速列出通话记录吗？

    如果在选项中启用了“读取通话记录”和“任意拼音”，输入 # 号，列出所有通话记录。输入 moshenren 或 msr，则出现所有陌生人的通话记录。
    　
  6. 不在联系人列表中的电话号码，比如 10086 或者 110，怎么拨打？

    使用屏幕上的“通话”按钮即可拨打输入的电话号码；或者，如果没有启用“拦截拨号按键”，使用拨号按键（即设备上的硬件按键）也是拨打输入的号码（而不是 CeleDial 的条目）。
    　
  7. 如何自定义拼音编码？

    如果你认为某个拼音编码有误，或者需要自定义拼音编码转换，可以建立一个 UNICODE 文本文件，命名为 CeleDial.txt 并放在与 CeleDial 同一文件夹下。每行一种拼音编码，格式为“[编码][空格][汉字1][汉字2]...[汉字n]”，例如“zzz 中好”（不包括引号），则“中”字和“好”字的拼音编码就变成了“zzz”。
