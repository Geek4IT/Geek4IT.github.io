一、简介
CeleTask 是 Windows Mobile 上使用的任务、进程和工具管理器。它可以帮助您全方面地掌握设备的运行状况，便捷地管理系统中运行的任务和进程，使 Smartphone 和 Pocket PC 真正具有多任务功能。

CeleTask 是一个免费软件， 您可以不受限制地使用它的全部功能。如果您愿意为使用它而付出任何回馈，请参看“捐助”。

二、功能
  1. 任务管理：包括新建任务、关闭任务、切换任务和查看任务窗口属性。
    　
  2. 进程管理：包括终止进程、查看进程信息。
    　
  3. 工具管理：完全可扩展定制的工具箱，可以为任意工具设置自动启动，创建快捷方式，或替换某些系统功能键。
    　
  4. 定制热键：允许用户为任何一个命令设置热键。支持后台长按热键唤出 CeleTask。
    　
  5. 内置工具：可以查看电量、电压、电流和温度等最详尽的电源状态信息；查看内存、内部存储和存储卡的使用状况；并具有清理内存、关闭系统和重新启动等功能。
    　
  6. 命令支持：所有命令均支持命令行参数操作，用户可以通过命令行参数执行单个或批量命令。
    　
  7. 完美支持：一个文件同时支持 Smartphone 2003、Pocket PC 2003、Windows Mobile 5.0 Smartphone 和 Windows Mobile 5.0 Pocket PC，支持 VGA/QVGA 和标准屏幕尺寸。
    　
  8. 迅捷易用：短小精干，载入快速！所有功能，尽在一个不到 60 K 的文件中。

三、安装
  1. 安装：把 CeleTask 复制到设备上的任何位置（比如“\Storage\Program Files\CeleTask”目录中）。如果不想在设备上查看帮助，只需要复制 CeleTask.exe 即可。如果需要，您可以在 CeleTask 工具箱中，编辑“任务管家”工具条目，为 CeleTask 创建快捷方式、设置自动运行或者关联到某些系统功能键上（请参考“工具”对话框）。
    　
  2. 卸载：执行 菜单 -> 卸载 命令，即可删除 CeleTask 在注册表中的设置（注意：工具列表也会恢复到默认状态）。卸载 命令不会删除可执行文件，也不会取消在工具对话框进行的替换或创建操作。

四、菜单
  1. 查看：包含视图操作命令。

    任务：切换到任务视图。
    进程：切换到进程视图。
    工具：切换到工具视图。
    刷新：刷新当前视图和状态栏信息。
    前视图：切换到前一个视图。
    后视图：切换到后一个视图。
    上一页：向上翻页。
    下一页：向下翻页。
    　
  2. 工具：包含工具视图中的工具条目（如果更改工具视图内的条目，必须在下一次运行的时候才能更新此菜单）。

    重新启动：重新启动设备。
    关闭系统：关闭系统电源。
    清理内存：关闭所有任务，并向所有窗口发送释放内存消息。
    电源信息：查看电源的详细信息和系统运行时间。
    存储信息：查看内存和存储的使用情况。
    　
  3. 选项：包含选项设置命令。

    显示所有：显示所有条目，即使是排除的条目。
    显示序号：显示条目序号。
    显示图标：显示条目图标。
    显示警告：为某些危险操作显示警告，以提醒用户谨慎执行操作。
    显示状态栏：在视图中显示状态栏信息，包括主要电池的电量和内存使用情况。
    转到后退出：执行“转到”任务命令后，退出 CeleTask。
    热键：请参考“热键”对话框。
    卸载：请参考“卸载”。
    　
  4. 帮助：包含帮助信息。

    帮助：显示帮助信息。
    电邮：给作者写电子邮件（欢迎给作者写电子邮件提出意见、建议或问题。此功能需要连接互联网）。
    网站：访问作者网站（即 http://www.yonsm.net/CeleTask。此功能需要连接互联网）。
    关于：请参考“关于”对话框。
    　
  5. 隐藏：隐藏 CeleTask。可以长按后台唤出热键或再次执行 CeleTask 恢复显示。
    　
  6. 退出：保存设置，退出程序。

五、视图
  1. 任务：列出了当前正在运行的任务。

    转到：切换到选定任务的活动窗口。
    属性：查看任务信息。
    关闭：关闭选定任务。如果在命令行参数中使用此命令，将关闭特定窗口或进程，如“CeleTask.exe 1042 Internet Explorer”或“CeleTask.exe 1042 IExplore.exe”。
    排除：切换选定任务的是否在列表中显示（如果在选项中选择了“显示所有”，排除的任务也会用灰色条目显示在列表中）。
    关闭所有：关闭列表中所有的任务。如果在命令行中使用此命令“CeleTask.exe 1044”，将关闭所有任务后整理内存并退出。
    关闭其它：关闭列表中除选定项目之外的任务。
    新建任务：请参考“新建任务”对话框。
    　
  2. 进程：列出了当前正在运行的进程。

    模块：查看选定进程的模块信息。
    属性：查看进程信息。
    终止：强行终止选定进程（请对您要进行的操作负责，不要随意终止进程，否则可能导致系统不稳定）。
    排除：请参考关于“排除”的描述。
    　
  3. 工具：列出了用户可定制的常用工具列表。此列表 将来 可能用于动态生成可扩展的工具菜单，并支持在任意位置弹出工具菜单。

    执行：执行选定工具。
    编辑：请参考“工具”对话框。
    删除：删除选定工具。
    排除：请参考关于“排除”的描述。
    向上移动：将条目往上移动。
    向下移动：将条目往下移动。
    添加工具：请参考“工具”对话框。

六、对话框
  1. 新建任务：类似 Windows PC 中的“运行”功能，允许用户指定参数运行程序或快捷方式。
    　
  2. 工具：允许用户添加或编辑工具条目。

    名称：工具显示名称。
    命令：可执行文件或快捷方式的路径。
    参数：可选命令行参数。

    替换右键按钮：用工具命令替换主屏幕右键按钮（通常情况下 此按钮显示为“联系人”）。
    替换开始菜单：用工具命令替换开始菜单（不建议使用）。
    替换快速列表：用工具命令替换快速列表（如果您不希望 CeleTask 常驻内存，建议替换快速列表，以便在任何位置方便唤出 CeleTask。快速列表关联的系统功能键，不同设备并不相同，请自行确定）。
    替换语音便笺：用工具命令替换语音便笺（一般说来，语音便笺默认关联到录音键。替换后，即使在开始菜单中执行“语音便笺”快捷方式，也会执行当前编辑的工具命令）。
    随系统自启动：在系统启动文件夹中创建工具命令的快捷方式。
    创建快捷方式：在开始菜单中创建工具命令的快捷方式。

    以上所有替换操作，均可取消勾选，恢复到初始状态。替换操作在 Pocket PC 平台上可能不起作用。
    　
  3. 热键：查看或设置命令热键。

    用向上或向下键可以在条目中导航（向上和向下也应用于主界面列表导航，不能设置为热键键）。
    在选定条目上可以按下期望的热键，再次按下则清除热键。
    支持长按热键，请在条目中长按热键达到相应的次数即可，但某些系统（比如后退、录音、音量+、音量-）键不支持长按。
    CeleTask 命令的热键为后台呼出热键（默认为长按 0），可以在后台运行 CeleTask 时恢复显示界面。正常运行时无需监视此热键消息，故不会导致任何性能损失。
    Smartphone 2003 操作系统中支持设置拨号键设置为热键（比如可以用此热键作为后台唤出热键），但 Windows Mobile 5.0 不支持。
    如果没有设置数字键为热键，按下数字键将激活相关条目。
    如果没有设置回车键为热键，按下回车键将激活选定条目。

    默认热键如下：

    星号：关闭所有任务；长按 关闭其它任务
    井号：任务属性/进程属性/编辑工具；长按 新建任务/添加工具
    向左：前视图；长按 电源信息
    向右：后视图；长按 存储信息
    后退：退出
    录音：显示所有
    音量+：向上翻页
    音量-：向下翻页
    数字0：刷新；长按 后台呼出 CeleTask
    　
  4. 关于：显示程序的版本、版权、许可、编译时间和作者信息。

七、命令

CeleTask 支持把所有的命令作为命令行参数（可以从 热键 对话框中查看菜单命令的标识），举例如下：

  • 显示工具视图：CeleTask.exe 1012
  • 显示电源信息，然后不显示主界面即退出：CeleTask 1053,1003
  • 显示进程视图，执行关于命令，再执行刷新操作：CeleTask.exe 1011,1033,1013

如果 CeleTask 正在运行，命令将发送给正在运行的 CeleTask 执行，否则由当前进程执行。

八、技巧
  1. 如何将状态栏显示在上方？

    编辑注册表键值“HKEY_CURRENT_USER\Software\CeleTask\Flag”，把原有数值加 1 变成奇数（如原来是 502，则改为 503）。
    　
  2. 如何导出工具设置？

    导出注册表“HKEY_CURRENT_USER\Software\CeleTask\”下所有以“Tool”开头的键值，需要时再导入即可。也可以通过此方法在 Windows PC 上快速编辑工具列表（请参照默认工具的键值）。
    　
  3. 如何避免不小心关闭了任务？

    CeleTask 在执行关闭任务时，首先发送 WM_CLOSE 消息给目标窗口，以期安全关闭任务。如果目标窗口不响应关闭命令，CeleTask 将强行关闭任务的进程。CeleTask 在关闭任务是没有任何警告，请谨慎确认要关闭任务，再执行关闭命令。

    另外，终止进程和删除工具命令允许设置命令提示（推荐设置“显示警告”选项），但同时也需要用户谨慎操作，以免造成不期望的后果。
    　
  4. 电源信息为何不准确？

    电源信息是使用系统函数获取的，在某些设备上可能不准确（比如某些机器上显示的温度都可以烧开水了），仅作参考之用。
    　
  5. 怎么把 CeleTask 关联到主屏幕右键菜单？

    先选中“显示所有”选项，在“工具”视图中，编辑“任务管家”工具条目属性，可以选择或取消右键菜单（此操作可以应用于给任何工具条目）。在主屏幕上执行右键按钮一次，或者重新启动系统 ，即可生效。注意：主屏幕右键菜单的命令不支持命令行参数，如果要启动时总是显示任务视图，需要先建立一个“CeleTask.exe 1010”的快捷方式（在开始菜单或任何位置均可），再在工具视图中把这个快捷方式加进来，然后设置这个快捷方式替换右键菜单。
    　
  6. 怎么 2.0 没有 1.7 方便、快速？如何提高运行速度？

    关于方便性，2.0 的其实设计更合理，如果不符合你的使用习惯，可以使用热键定制功能。2.0 默认显示项目图标，这将会降低程序的运行速度。取消“显示图标”和“显示状态栏”选项， 可以大大提高速度。
    　
  7. CeleTask 版本号是什么意思？

    以 2.0.2051.4212 例说明：2 为主版本号，表示重大升级；0 为次版本号，表示功能上增强升级 ，奇数则表示小规模升级；2051 为 Release 号，表示我在开发过程中使用 Release 方式构建的次数；4212 为 Build 号，表示开发过程中总共编译次数（用工具 BPP 自动增加的，可以在我的主页上下载）。每次不带 BETA 标记的公开发布都属于正式版本，如果主版本号和次版本号都未改变，则表示修改问题或小改进。请访问我的网站检查软件更新。
    　
  8. Smartphone 2003 下无法浏览获取文件路径，有解决方法吗？

    在 Smartphone 2003 中不支持浏览获取文件路径。不过，CeleTask 支持扩展 RFileShell 和 tGetFile 浏览获取文件名，优先选择顺序分别为 tGetFile、RFileShell、AygShell、CoreDll。Smartphone 2003 中可以使用 RFileShell 或 tGetFile 来实现此功能。其它系统系统中，也可以使用 tGetFile 和 RFileShell。
